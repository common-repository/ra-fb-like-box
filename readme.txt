RA-FB Like Box enables you to display the facebook page likes in your website.

== Description ==

With this plugin You can put the Facebook Likebox to Your Site to attact people to your website


== Installation ==

1. Download the plugin.
2. Upload to your blog (/wp-content/plugins/).
3. Activate it.
4. Click the RA-FB Like Box.
5. Fill in the options.
6. Then go to widget and drag and drop Rashid's Facebook Like Box in the area you want to display the plugin.
   Alternatively you can add shortcode to your post or page to display the likebox. Just copy and paste following 
   code on the page or post:
   [rashid_fb_lb/]

You're done!

Uninstalling is as simple as deactivating the plugin.
